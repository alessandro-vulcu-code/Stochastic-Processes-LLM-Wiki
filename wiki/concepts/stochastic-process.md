# Stochastic Process

**A stochastic process is a family of random variables $\{X_t\}_{t\in T}$ indexed by a set $T$.**

## Definition

For each $t\in T$, $X_t$ is a random variable taking values in a state space $S$. The index set may be discrete, such as $T=\mathbb{N}$, or continuous, such as $T=[0,\infty)$.

A process is fully specified by the joint distributions of every finite collection $\{X_t:t\in \bar T\}$.

## Key Properties

- The index set can represent time, space, or another ordered parameter.
- The state space can be discrete or continuous.
- If $T$ and $S$ are both discrete, the process is called a chain.
- If all $X_t$ have the same distribution, the process is stationary in the marginal sense used by Chapter 1.

## Intuition

A stochastic process is a random evolving object. One experiment produces one sample path; repeating the experiment gives another path.

- Number of arrivals up to time $t$.
