# Long Run Behaviour of Markov Chains

## Overview

This topic classifies Markov-chain states and classes according to accessibility, communication, periodicity, recurrence, and return times. The main objective is to determine when $P_{ij}^{(n)}$ converges, what it converges to, and how reducible or periodic structure changes the answer.
